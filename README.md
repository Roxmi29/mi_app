# Mi Proyecto con Reflex

## Descripción
Este proyecto es una práctica académica para aprender a usar el framework **Reflex** en Python.  
La aplicación muestra un título, un texto de bienvenida y un botón interactivo que cambia el mensaje al hacer clic.

## Requisitos
- Python 3.10 o superior
- Reflex 0.9.1
- Entorno virtual (`venv`)
